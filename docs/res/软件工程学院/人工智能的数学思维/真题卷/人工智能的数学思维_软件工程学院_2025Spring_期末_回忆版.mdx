# 考试信息整理

## 成绩构成

- **平时成绩**: 40%
- **期末成绩**: 60%

## 考试内容范围

- **重点**: 1956年达特茅斯会议后的内容。
- **未考内容**: 1956年之前的数学史。

## 考试题型与分值

1. **单选题**
   - 题量: 10题
   - 分值: 20分
   - 示例考点: 激活函数

2. **多选题**
   - 题量: 10题
   - 分值: 10分

3. **判断题**
   - 题量: 5题
   - 分值: 10分
   - 示例考点:
     - 题目1: "天下没有免费的午餐，所有优化算法在平均意义上表现相同"

4. **简答题**
   - 题量: 6题
   - 分值: 60分
   - 示例考点:
     1. **什么是“监督学习”?** 请简要说明它与“无监督学习”的主要区别，并各举一个典型应用场景。
     2. **通用近似定理的意义**，并说明激活函数在其中的作用。
     3. **卷积神经网络(CNN)的基本结构**，及其在图像处理中的作用原理。
     4. **深度学习中“前向传播”和“反向传播”的流程与作用**。
     5. **“从规则驱动到数据驱动”的范式转变**? 结合生活中的例子说明其意义。
     6. **人工智能发展的三个主要阶段**，当前所处阶段及其特点。

## 学习重点提示

- **概念重要性**: 概念知识是考试的核心，需要重点掌握。
- **发展史**: 发展史的内容仅在简答题中涉及，相对次要。
